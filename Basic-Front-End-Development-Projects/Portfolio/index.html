SKILLS & EXPERTISE

I've developed my skill-set to provide a suite of tools best suited to getting the job done efficiently and effectively

Frontend development

Whether you need a new website, updates to an existing application or design ideas; if it involves the web, I can help.

I write code that conforms to the W3C's web standards and semantic best-practices. Using technologies such as HTML5, CSS3, and javascript to change static artwork into interactive and fully functional interfaces, exactly the way you imaged them to be.

Full Stack development

To develop a truly immersive application requires knowledge of everything from the server and network layers, data modeling and business logic, and the user interface and experience.

I am experienced with every layer required to build and support your website or application. From the hardware side; servers, load balancing, network infrastructure, storage. Back-end; Databases, server-side code, data modeling. Front-end; User Interface design, client side code, cross browser support. Cyber Security; framework auditing, engineering, Ethical, policy auditing.

Responsive web development

The web is no longer just viewed from the desktop computer. Today the web is consumed on tablets, smart phones, and smart watches.

How will I insure that your website or application looks great no matter what size screen or device it is accessed from? I build using a responsive grid system to insure your website or application can adapt fluidly to the device that is accessing. This helps insure your site or application stay more future proof.
